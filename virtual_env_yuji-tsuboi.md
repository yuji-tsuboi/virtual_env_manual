| Homebrewのインストール     | Homebrew(ホームブルー)とはパッケージ管理システムの一つです。<br>パッケージ管理システムとは、実行ファイルや設定ファイルやライブラリを一括で管理しているシステムのことです。<br>これから行うPHPのインストールやMySQLの環境構築にはHomebrewを使います。<br>手順は以下の通りです。<br>インストールではターミナルを使用するのでターミナルアプリを開いてください。<br>HomebrewのインストールにはXcode(エックスコード)が必要です。まずは、Xcodeのインストールから行います。<br>XcodeとはAppleのソフトウェア開発ツールのセットです。Mac、iPhone、iPad、Apple WatchなどのAppleプラットフォーム用アプリケーションの作成に使用します。<br>下記のコマンドをターミナルに打った後に実行をしてください。<br><font color="DeepSkyBlue">``$ xcode-select --install``</font><br>ポップアップが出るのでインストールを進めて下さい。<br>また、既にインストールされているのであれば以下のような表示がターミナルで確認できます。<br>``xcode-select: error: command line tools are already installed, use "Software Update" to install updates``<br>既にインストールされている方は気にせず進んで下さい。<br>下記のリンクを開いてください。<br>[Homebrewの公式サイト](https://brew.sh/index_ja)<br>インストールの為のスクリプトが公式サイトに用意されているので、コマンドをターミナル上にコピペして実行しましょう。下ものは例だと思って下さい。<br><font color="DeepSkyBlue">``$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"``</font><br>インストールされると次の一文が何処で確認出来るはずです。<br>``==> Installation successful!``<br>確認できたら最後の段階に進みます。<br>最後は、インストールされているか下記のコマンドを実行して確認しましょう。<br><font color="DeepSkyBlue">``$ brew -v``</font><br>上記はHomebrewのバージョンを確認する為のコマンドです。以下のような表示が出ていればOKです。<br>``Homebrew 2.7.1``<br>``Homebrew/homebrew-core (git revision f3975; last commit 2021-01-03)``<br> ``Homebrew/homebrew-cask (git revision 2a0db; last commit 2021-01-03)``<br>これでHomebrewのインストールは完了です。 |

| Pathを通す             | PathとはファイルやディレクトリのPC内での場所のことです。<br>通常PHPはインストールされたディレクトリからしか使用できませんが、Pathを設定することでどのディレクトリからでも操作可能になります。<br>このPathを設定する作業を「Pathを通す」と表現します。<br>実際にPathを通してみます。<br>ターミナルで先ほど、PHPをインストールした際に表示された部分に下記のようなものがあります。<br>``If you need to have php@7.3 first in your PATH run:``<br>``echo 'export PATH="/usr/local/opt/php@7.3/bin:$PATH"' >> /Users/tsuboiyuuji/.bash_profile``<br>``echo 'export PATH="/usr/local/opt/php@7.3/sbin:$PATH"' >> /Users/tsuboiyuuji/.bash_profile``<br>見つけることが出来たら``echo``から始まるそれぞれの行を別々にコピペして実行しましょう。<br>必ず自身のターミナルからコピーペーストして実行して下さい。下のものは例です。<br><font color="DeepSkyBlue">``$ echo 'export PATH="/usr/local/opt/php@7.3/bin:$PATH"' >> /Users/tsuboiyuuji/.bash_profile``</font><br><font color="DeepSkyBlue">``$ echo 'export PATH="/usr/local/opt/php@7.3/sbin:$PATH"' >> /Users/tsuboiyuuji/.bash_profile``</font><br>次に設定を反映する為、``~/.bash_profile``を読み込みます。ターミナルを再起動しても設定は反映されます。<br><font color="DeepSkyBlue">``$ source ~/.bash_profile``</font><br>最後にしっかりと反映されているか``which``というコマンドを使って確認します。<br><font color="DeepSkyBlue">``$ which php``</font><br>下記の様なPathが帰ってきたらOKです。<br>``/usr/local/opt/php@7.3/bin/php``<br>これでPathを通す作業は完了です。                                                                                                                                                                                                                                                                                                                                       |

| 仮想環境内PHPインストール      | 仮想環境内にPHPをインストールします。<br>今回もyumとsudoを利用してインストールしていきます。<br>以下の手順でインストールしましょう。<br>まず、下準備としてPHPをインストールする為にEPELと言うリポジトリを下記のコマンドで有効にします。<br><font color="DeepSkyBlue">``$ sudo yum -y install epel-release``</font><br>これで``Remi``がインストール可能になりました。<br>次に、PHPインストールの準備としてRemi(レミ)をインストールします。下記のコマンドでインストールします。<br><font color="DeepSkyBlue">``$ sudo wget rpms.famillecollet.com/enterprise/remi-release-7.rpm``</font><br>``wget``(ダブルゲット)とはサーバーから指定したファイルを取ってくる事ができ、再帰的なダウンロードが可能なコマンドです。<br>その次にパッケージを更新します。<br><font color="DeepSkyBlue">``$ sudo rpm -Uvh remi-release-7.rpm``</font><br>PHPをと合わせて拡張モジュールを``Remi``を使ってインストールします。<br><font color="DeepSkyBlue">``$ sudo yum -y install --enablerepo=remi-php72 php php-pdo php-mysqlnd php-mbstring php-xml php-fpm php-common php-devel php-mysql unzip``</font><br>最後にバージョンを確認し出来ていればOKです。<br>これでPHPのインストールは完了です。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 仮想環境内Composerインストール | Composerをインストールしていきます。<br>まずはインストーラーをダウンロードしていきます。<br><font color="DeepSkyBlue">``$ php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"``</font><br>次に、ダウンロードしたインストーラーを実行します。<br><font color="DeepSkyBlue">``$ php composer-setup.php``</font><br>インストールしたインストーターを削除します。<br><font color="DeepSkyBlue">``$ php -r "unlink('composer-setup.php');"``</font><br>次にCentOSのどこからでも使えるようにします。<br><font color="DeepSkyBlue">``$ sudo mv composer.phar /usr/local/bin/composer``</font><br>最後に、バージョンを確認します。確認ができたら大丈夫です。<br><font color="DeepSkyBlue">``$ composer -v``</font><br>これでComposerのインストールは完了です。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| Laravelアプリケーションのコピー | 仮想環境内に、作成したLaravelアプリケーションをコピーします。<br>ゲストOSにいるならホストOSに下記のコマンドで移動します。<br><font color="DeepSkyBlue">``$ exit``</font><br>次に、Vagrantの作業用ディレクトリに下記のコマンドで移動します。<br><font color="DeepSkyBlue">``$ cd markdown_lesson``</font><br>次の一文でLaravelアプリケーションをコピーします。<br><font color="DeepSkyBlue">``$ cp -r laravel_lessonまでの絶対パス ./``</font><br>Vagrantの作業用ファイルに``laravel_lesson``のディレクトリがコピーされた事を確認して下さい。<br>確認ができたら完了となります。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| 仮想環境内MySQLのインストール    | Laravelアプリケーションを操作する為にMySQLのインストールをしていきます。<br>以下のコマンドを実行して公式リポジトリ情報ファイルのインストールを行います。<br><font color="DeepSkyBlue">``$ sudo wget https://dev.mysql.com/get/mysql57-community-release-el7-7.noarch.rpm``</font><br>次に、インストールしたパッケージファイルを更新します。<br><font color="DeepSkyBlue">``$ sudo rpm -Uvh mysql57-community-release-el7-7.noarch.rpm``</font><br>rmpとは、RPM（Red Hat Package Manager）パッケージ”を扱うことができるパッケージ管理コマンドです。<br>MySQLをインストールしていきます。<br><font color="DeepSkyBlue">`` $ sudo yum install -y mysql-community-server``</font><br>バージョンをします。<br><font color="DeepSkyBlue">``$ mysql --version``</font><br>バージョンが確認できたら完了となります。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 仮想環境下MySQLパスワード設定   | rpmでMySQLをインストールした場合、パスワードの設定をする必要性が出てきます。<br>以下の手順で変更していきます。<br>まずは、簡易的なパスワードを作成できるようにMySQLの設定ファイルに編集を加えていきます。<br><font color="DeepSkyBlue">``$ sudo vi /etc/my.cnf``</font><br>次の文を確認して下さい。<br>``socket=/var/lib/mysql/mysql.sock``<br>この文の下に1行追加していきます。<br>``validate-password=OFF``これで編集は以上です。<br>その次に、サーバーの再起動を行います。<br><font color="DeepSkyBlue">``$ sudo systemctl restart mysqld``</font><br>今現在のパスワードを確認します。<br><font color="DeepSkyBlue">``$ sudo cat /var/log/mysqld.log ｜grep 'temporary password'``</font><br>出力された``:``後のものが現在のパスワードです。<br>MySQL内での操作をしていきます。<br><font color="DeepSkyBlue">``$ mysql -u root -p``</font><br><font color="DeepSkyBlue">`` $ Enter password:確認したパスワード``</font><br><font color="DeepSkyBlue">``mysql > set password = 変更したいパスワード``</font><br>これでパスワードの設定は以上になります。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| 仮想環境内データベース作成            | MySQLの諸々の設定が終わったのでデータベースを作成していきます。<br>MySQL内で下記のコマンドを使用してデータベースの作成を行います。<br><font color="DeepSkyBlue">``mysql > create database laravel_lesson``</font><br>最後にLaravelアプリケーションでのDBの接続情報の変更をしていきます。<br>laravel_lessonディレクトリ下の``.evn``ファイルを編集していきます。``DB=PASSWORD=``に設定したパスワードを入力します。<br><font color="DeepSkyBlue">``DB_PASSWORD=入力したパスワード``</font><br>次のコマンドでマイグレーションファイルを実行します。<br><font color="DeepSkyBlue">``$ php artisan migrate``</font><br>これで仮想環境内でのデータベースの作成は完了です。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| 仮想環境内Nginxインストール    | WebサーバーであるNginx(エンジンエックス)をインストールします。<br>まず、root権限で新たにファイル``/etc/yum.repos.d/nginx.repo``を追加します。<br><font color="DeepSkyBlue">``$ sudo vi /etc/yum.repos.d/nginx.repo``</font><br>``vi``コマンドとはLinuxの標準のエディタを起動するコマンドです。<br>作成されていないファイルをパスとして設定すると自動で作成してくれます。<br>``vi``ではモードが2つあり、ノーマルモードとインサートモードがあります。<br>切り替えは``escキー``とキーボードの``i``で行えます。<br>ノーマルモードは編集は行えませんが、保存や削除ができます。<br>対してインサートモードは編集処理が可能です。<br>起動したらまず、<font color="DeepSkyBlue">``i``</font>でインサートモードに切り替えます。その後に、以下の記述をコピーペーストして下さい。<br><font color="DeepSkyBlue">``[nginx]``</font><br><font color="DeepSkyBlue">``name=nginx repo``</font><br><font color="DeepSkyBlue">``baseurl=https://nginx.org/packages/mainline/centos/\$releasever/\$basearch/``</font><br><font color="DeepSkyBlue">``gpgcheck=0``</font><br><font color="DeepSkyBlue">``enabled=1``</font><br>コピーペーストが出来たら<font color="DeepSkyBlue">``escキー``</font>でノーマルモードに切り替えて、<font color="DeepSkyBlue">``:wq``</font>で保存をかけて下さい。<br>ファイル追加後、次のように``yum install``コマンドでインストールできます。<br><font color="DeepSkyBlue">``$ sudo yum install -y nginx``</font><br> バージョンを確認します。<br><font color="DeepSkyBlue">``$ nginx -v``</font><br>バージョンが確認できたら完了となります。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| nginx設定ファイル編集       | Laravelを動かす為の記述をNginxの設定ファイルに書いていきます。<br>次のコマンドで設定ファイルを開きます。<br><font color="DeepSkyBlue">``$ sudo vi /etc/nginx/conf.d/default.conf``</font><br>開いたら次の部分を編集していきます。<br>``server_name  192.168.33.19;``に編集<br>``root /vagrant/laravel_lesson/public;``を追加<br>``index  index.html index.htm index.php;``を追加<br>``root   /usr/share/nginx/html;``と``index  index.html index.htm;``をコメントアウト<br>``location /``内に``try_files $uri $uri/ /index.php$is_args$args;``を追加<br> ``location ~ \.php$``と、その中のroot以外のコメントをコメントアウトする<br>``location ~ \.php$``の中の``fastcgi_param``の部分を``SCRIPT_FILENAME  /$document_root/$fastcgi_script_name;``に編集<br>これで編集は完了です。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| パーミッションの設定          | パーミッションとは操作権限のことです。<br>今のままだと十分な操作権限がない為Laravelのアプリケーションを操作する事ができません。<br>以下のコマンドで操作権限を付与します。<br><font color="DeepSkyBlue">``$ cd /vagrant/laravel``</font><br><font color="DeepSkyBlue">``$ sudo chmod -R 777 storage``</font><br>次に、操作できる権限を持てるユーザーを変更します。<br><font color="DeepSkyBlue">``$ sudo chown vagrant:vagrant /var``</font><br>これでパーミッションの設定は終了です。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
